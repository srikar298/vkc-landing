# Phase 2: The Economic Engine (PM Perspective)

**Target Timeline**: Months 3-4

## Objective
Provide tangible economic empowerment by launching the ecosystem directories.

## Strategic Value
* **Monetization Potential**: Core utility of the app connecting buyers directly with artisans.
* **The "Cold Start" Problem**: Aggressively seed the directory using Phase 1 Membership leads.

## User Personas (The Network Core)
The Phase 2 expansion is driven by 8 key user categories, each fulfilling a specific role in the community ecosystem:

| Persona | Strategic Role | Value Proposition |
| :--- | :--- | :--- |
| **Public Guest** | Visitor | Exposure to heritage; potential for brand advocacy. |
| **Digital Artisan** | Base Member | Basic digital identity; gateway to the professional hub. |
| **Verified Member** | Trusted Member | Access to sensitive matrimony and networking data. |
| **Professional Expert**| Mentor/Consultant | Offers specialized skills (Medical, Legal, IT) to the community. |
| **Community Official**| Leader | Provides institutional credibility and government linkage. |
| **Matrimony Candidate**| Seeker | High-trust matchmaking within the community. |
| **Mentor / Mentee** | Educator | Knowledge transfer and heritage preservation training. |
| **Donor (Patron/Gold)**| Benefactor | Funds community initiatives; gains prestige and VIP access. |

## Feature Evaluation & Edge Cases
* **Trust & Quality**: Implement Yelp-style ratings. Only authenticated users can review.
* **Privacy Controls**: Robust privacy toggles for Professionals ("Hide Phone Number").
* **Verification Waterfall**: Only `Verified Members` can access the Matrimony search to prevent data harvesting.
