# Phase 3: Content & Community Support (UI/UX Perspective)

## Information Architecture
* Consolidate features under `Media & Resources`. Create a personalized dashboard feed.

## UX Design & Aesthetics
* **Reading Experience**: Adjustable font sizes, dark mode. Handle Telugu fonts with generous line-height and container padding.
* **Video Gallery**: Lazy load YouTube embeds with high-quality skeleton loaders.
* **Document Management**: Inline PDF viewers and clear "Download [PDF, X MB]" buttons.
